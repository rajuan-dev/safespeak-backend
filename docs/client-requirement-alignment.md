## Current MVP Blockers

1. Complete final manual sign-off for consent gates, Quick Exit cleanup, heatmap suppression, and RAG citation fallback.
2. Run `npm run rag:ingest:official:mvp`, complete client/legal review in the admin Knowledge Sources page, approve eligible sources, then confirm `npm run rag:readiness -- --fail-on-not-ready` passes before claiming jurisdiction-specific legal coverage.
3. Configure real partner delivery credentials before any UI says a report was actually sent to an agency.
4. Complete final real-device voice smoke checks on the public AI conversation/report page: microphone permission, tap-to-record, stop-record, Safari support, and noisy-audio fallback errors. Automated e2e now covers consent recovery, transcript-to-conversation, autoplay fallback, replay, and Stop Recording.
5. Keep mobile-native/offline/AU-cloud/security-signoff, subscription/fee management, and landing-page CMS/refresh items in a separate rollout tracker.

## Admin Role Flow

| Role               | Backend access                                                                                                                                                                          | Admin UI flow                                                                                             |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| Super Admin        | Full admin access, users, admin creation, audit logs, privacy requests, data protection, content, integrations, analytics                                                               | Sees all role-relevant admin navigation and can create scoped admin users.                                |
| Content Admin      | Taxonomies, cultural profiles, RAG knowledge sources, AI/content overview, language packs, microeducation, content resources, media assets, resource library, platform content settings | Lands in content/education flow; sidebar hides user, audit, integration-only, and analytics-only areas.   |
| Integrations Admin | Destinations, submission templates, report delivery monitor, support service registry, warm referral admin queue                                                                        | Lands in service destinations/support flow; sidebar hides content, user, audit, and analytics-only areas. |
| Analytics Viewer   | Analytics overview, heatmaps, trends, categories, language analytics, incident intelligence                                                                                             | Lands in incident insights; sidebar hides write/admin management areas.                                   |
| Legacy `admin`     | Authenticated admin shell only; sensitive route handlers require one of the scoped roles above                                                                                          | Dashboard/profile only in UI unless upgraded by Super Admin to a scoped role.                             |
