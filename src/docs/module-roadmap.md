# Module Roadmap

This foundation intentionally implements only health checks.

Future module candidates:

- Auth
- Anonymous Session
- RBAC
- Consent
- Profile
- Reports
- Evidence Vault
- AI legal-information support
- RAG
- ScamShield
- Routing
- Admin
- Analytics
- Privacy Compliance

Each module should be added only when explicitly requested and should live under `src/modules/<module-name>`.
